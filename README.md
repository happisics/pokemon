# Pokemon

"poketball"
"pokemon.exe"
둘을 같은 위치에 저장해 두어야함!!


포켓몬이 등장하며 원하는 포켓몬을 poketball에 gif형식으로 저장하여 끌어낼 수 있다.

pokemon.exe를 실행한 뒤 컴퓨터를 유영하는 포켓몬을 가지고 놀자.

포켓몬이나 포켓볼에 마우스를 가져다 대고 스크롤을 하면 크기가 바뀌고, 마우스로 끌어서 이동할 수 있다.

(스크롤 클릭시 원래크기로 복구됨)



포켓볼을 포켓몬 위에 가져다대면 일정확률로 포획되며 잡힌다.



종료하기위해서는 모든 포켓몬을 잡아야하니 너무 많은 포켓몬을 넣지 말기를 바란다.

(물론 강제종료해도 종료는 되니 참고)
